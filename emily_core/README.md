# Emily Core

Core functionality for Emily Tools - unified memory store, database management, and data models.

## Installation

This package is installed as a dependency of the main emily-tools package.

## Usage

```python
from emily_core import UnifiedMemoryStore, DatabaseManager, MemoryEntity
```

## Features

- **UnifiedMemoryStore**: Centralized memory management system
- **DatabaseManager**: Database operations and management
- **MemoryEntity**: Data models for memory entities
- **MemoryRelation**: Data models for memory relations
- **MemoryContext**: Data models for memory contexts

## Development

To install in development mode:

```bash
pip install -e .
```
